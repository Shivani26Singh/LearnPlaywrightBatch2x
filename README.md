# LEARNPLAYWRIGHTBATCH2X

## Overview

This repository contains materials for a JavaScript + Playwright + AI course. It covers:

- JavaScript fundamentals and concepts
- Playwright automation basics for browser testing
- AI-related learning and examples as part of the course content

## Repository Structure

- `chapter_01_Basics/` - Basic JavaScript exercises, environment verification, and introductory runtime examples
- `chapter_02_Javascript_Concepts/` - Core JavaScript variable usage and basic scripting examples
- `chapter_03_Identifier_Literals/` - JavaScript identifier rules, naming conventions, comments, and identifier examples

## Chapter Details

### `chapter_01_Basics/`
- `01_Basics.js` - prints a hello message to confirm JavaScript execution
- `02_JS.js` - demonstrates variable declaration and logging values
- `03_JS_Verify_Setup.js` - checks Node.js platform, architecture, and version
- `04_HotCode.js` - shows a simple function loop to illustrate repeated execution and performance testing

### `chapter_02_Javascript_Concepts/`
- `05_JS_Basics.js` - explores variable reassignment and console output for basic JavaScript behavior

### `chapter_03_Identifier_Literals/`
- `06_Identifier_Rules.js` - valid identifier examples using letters, underscores, dollar signs, and mixed digits
- `07_Identified_Part2.js` - naming conventions including camelCase, PascalCase, snake_case, SCREAMING_SNAKE_CASE, and Hungarian notation
- `08_Comments.js` - single-line, multi-line, and JSDoc-style comments
- `js_identifer_rules.js` - detailed identifier rules, including reserved keyword restrictions, case sensitivity, Unicode support, and invalid name examples
- `VS_Code_keyboard_shortcut_mac.md` - macOS keyboard shortcuts for VS Code
- `VS_Code_keyboard_shortcut_windows.md` - Windows keyboard shortcuts for VS Code

## Getting Started

### Prerequisites

- Node.js installed
- npm or yarn available
- Playwright installed globally or locally in a project

### Suggested Setup

1. Open the repository in your code editor.
2. Install Playwright if needed:

```bash
npm install playwright
```

3. Optionally install other dependencies if a project is added later.

### Run Playwright Tests

If the course is expanded to include a test project with Playwright scripts, use:

```bash
npx playwright test
```

or a direct script command:

```bash
node <script-file.js>
```
